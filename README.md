## Overview
This project demonstrates *end-to-end data analysis* using *Python, SQL, and Pandas*.  
A structured sales dataset is analyzed to answer real-world business questions such as:
- What drives revenue?
- Which products and regions perform best?
- Is revenue driven by price or quantity?

The project reflects *industry-style analytical thinking*, not just query writing.

---

## Dataset
The dataset (sales_data.csv) contains transactional sales data with the following fields:
- order_id
- order_date
- product
- category
- quantity
- price
- region
